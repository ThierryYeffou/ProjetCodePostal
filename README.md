# ProjetCodePostal

Bonjour,


Pour ce projet j'ai utilisé les techniques web suivants:

-> Php, HTML, CSS.
-> Base de Données mysql.
-> Serveur local apache
-> Serveur FTP FileZilla qui me permet de passer mes fichiers sur le serveur en ligne
-> Hébergeur optionnel fait Hostinger

Comme base de données j'ai utilisé PHPmyadmin. En effet vu qu'il existait un fichier excel en ligne qui dispose de tous les codes postaux des différentes 
villes de France. J'ai fait le choix de lire le fichier et de le stocker dans ma base de donnée. Par contre une possibilité que j'avais 
aussi était d'utiliser un api car j'ai fait des recherches et l'api des codes postaux villes existe bien j'ai bien les données 
sur le format jsons mais je n'ai pas fait ce choix.

Donc pour ce projet:
il était question pour moi de mettre mes dossiers dans le serveur wamp vu que j'ai tout d'abord travaillé en localhost.
Ensuite dans le but d'héberger pour vous faire visualiser, j'ai utilisé un serveur gratuit en ligne.
et j'ai changé les données de connexion de ma base locale avec celles de ma base en line comme vous allez constater dans le code source documenté.
